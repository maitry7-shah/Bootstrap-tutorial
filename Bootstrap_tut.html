The Bootstrap grid system has four classes: xs (for phones) sm (for tablets) md (for desktops) lg (for larger desktops)

Grid columns are created by specifying the number of 12 available columns you wish to span
For example: Three equal columns would use three .col-sm-4 || Two equal columns would use two .col-sm-6 || Twelve equal columns would use twelve .col-sm-12

BS Image Shapes




Bootstrap adds shapes to images by adding a class to the <img> tag


Responsive images automatically adjust to fit the size of the screen. You can create responsive images by adding an .img-responsive class to the tag. The image will then scale nicely to the parent element.

Try it yourself:  <img src="img/giphy.gif" class="img-rounded" alt="giphy" width="200" height="236">

Class Description
.img-responsive Makes an image responsive
.img-rounded Adds rounded corners to an image
.img-circle Shapes the image to a circle
.img-thumbnail Shapes the image to a thumbnail
